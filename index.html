Esta pagina esta destinada a facilitar el calculo de las areas de las figuras mas comunes. BIENVENIDO.

Area del Cuadrado

El área de un cuadrado es la medida de su superficie interior. Se calcula multiplicando la longitud de un lado por sí misma

Triangulo

El área de un triángulo se calcula multiplicando la base por la altura y dividiendo el resultado entre 2

Circulo

El área de un círculo es la medida de la superficie que abarca su interior. Se calcula usando la fórmula: A = π x r^2

Cubo

El área de un rombo se calcula multiplicando las longitudes de sus dos diagonales y dividiendo el resultado entre 2.
